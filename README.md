# Budget-pro

A simple, responsive web application for tracking personal expenses and income with file import/export functionality.

## Features

- **Add Transactions**: Track both expenses and income with title, amount, category, date, and description
- **Dashboard**: Real-time overview of total balance, income, and expenses
- **Transaction History**: View all transactions with filtering options
- **File Operations**: Export data to JSON file and import from JSON file
- **Visual Charts**: Doughnut chart showing expense distribution by category
- **Responsive Design**: Works on desktop and mobile devices
- **Local Storage**: Data persists in browser between sessions
- **Category Management**: 8 predefined categories with color coding

## How to Use

1. **Add a Transaction**:
   - Fill in the form on the left side
   - Select transaction type (expense/income)
   - Choose category
   - Enter amount and date
   - Click "Add Transaction"

2. **View Transactions**:
   - All transactions appear in the history section
   - Use filters to view specific types or categories
   - Click the trash icon to delete a transaction

3. **Export Data**:
   - Click "Export to File" to download all transactions as a JSON file
   - The file is saved with timestamp in the filename

4. **Import Data**:
   - Click "Import from File"
   - Select a previously exported JSON file
   - Data will be merged with existing transactions

5. **View Statistics**:
   - Check the dashboard for totals
   - View the chart for expense breakdown by category
